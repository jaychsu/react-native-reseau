react-native-reseau
======

## Usage

### Sample Data

```javascript
[
  { g: 'Group A', s: require(./test.png), t: 'a' },
  { g: 'Group A', s: 'http://www.123.com', t: 'b' },
  { g: 'Group B', s: '', t: '' },
  { g: 'Group A', s: '', t: '' },
  { g: 'Group B', s: '', t: '' },
  { g: 'Group C', s: '', t: '' },
  { g: 'Group A', s: '', t: '' },
  { g: 'Group C', s: '', t: '' },
  { s: '', t: '' },
  { s: '', t: '' },
  { g: 'Group C', s: '', t: '' },
  { g: 'Group C', s: '', t: '' },
  { g: 'Group B', s: '', t: '' },
  { s: '', t: '' },
  { s: '', t: '' },
]
```

About properties:

- `g`: `string` group name
- `o`: `number` order
- `p`: `number` page
- `s`: `string | number` src of thumbnail
- `t`: `string` title
